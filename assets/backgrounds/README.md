# Background Images

Place one JPEG per theme here. Each file must match its theme name exactly:

| File | Theme |
|------|-------|
| `beach.jpg` | Beach |
| `mountain.jpg` | Mountain |
| `desert.jpg` | Desert |
| `wooded.jpg` | Wooded |
| `underwater.jpg` | Underwater |

**Recommended specs:** 1080×1920px minimum, JPG quality 85. The image is blurred
at runtime (blur radius varies per theme) so extreme sharpness is not required,
but strong color and composition still show through the blur.
